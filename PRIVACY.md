# Privacy Policy

Prompt Auto-Dismiss for ChatGPT does not collect, store, transmit, sell, or share personal data.

The extension runs only on ChatGPT pages listed in its manifest. It looks for matching on-page prompt text and clicks a visible "Got it" button when the prompt matches the extension's rules.

The extension uses Chrome's `storage` permission only to save the user's local settings, such as whether auto-dismiss is enabled, the matching style, and the minimum delay between automatic clicks.

No browsing history, ChatGPT conversations, account information, page content, analytics, or usage data is sent to the developer or any third party.
