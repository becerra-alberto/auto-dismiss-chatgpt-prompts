# Prompt Auto-Dismiss for ChatGPT

## Short Description

Automatically dismiss selected ChatGPT prompts when they appear.

## Detailed Description

Prompt Auto-Dismiss for ChatGPT quietly clicks the visible "Got it" button on matching ChatGPT prompts, starting with the "Too many requests" message. It includes a toolbar popup where you can turn the behavior on or off, choose stricter or more flexible prompt matching, and set the minimum delay between automatic clicks.

The extension is intentionally narrow:

- It only runs on `chatgpt.com` and `chat.openai.com`.
- It does not read, collect, transmit, or sell user data.
- It does not bypass rate limits or change account access.
- It only dismisses matching prompts after they are shown.

## Privacy Practices

Single purpose: dismiss selected ChatGPT prompts when they appear.

Data usage: no personal data collection.

Permissions:

- `storage`: saves the user's local extension settings.

Host access:

- `https://chatgpt.com/*`
- `https://chat.openai.com/*`
